Brackets - a perfect IDE for beginners

Why do I use Brackets and why do I reccommend it?

October 16, 2019 Lazar RistiÄ‡


If you are a beginner and only getting into Web Design and Web Development, you are probably looking for an IDE that will do you right. Eventually you end up downloading VS Code or Atom, or some of the JetBrains products. I'm not saying they are not good, here I am talking about the first Web Design steps. This is why I pick Brackets over any of the mentioned above.

Brackets' simplicity is what made me download it and I did not regret it one bit. It's fast, has awesome options and doesn't offer unnecessary stuff. I will go through some of these cool options:

  1. Inline CSS editing

    2. Since this is beginner-oriented, I won't be getting into SCSS options with the inline editor. Brackets has an option where you can select an ID or a Class, or any element in the HTML and press CTRL+E or Command+E and an inline editor will pop up below showing the CSS and SCSS properties of a selected element. This way you won't be switching tabs all the time and messing up your workflow.

    As you can see I selected an id called footer and an editor popped up below showing it's CSS properties.


  2. Live Preview

    3. I cannot stress enough how useful this option was for me. Live Preview gives you an opportunity to change the code in Brackets and with Live Previews turned on, it will automatically change in the browser (locally, ofcourse). You don't have to open a file everytime or refresh or anything, just change the code in the editor and the webpage will too be changed. It is located in the upper-right corner of your screen:


    This Lightning icon is the Live Preview, it's orange when it's On and grey when it's Off.


  3. Themes, multi-select, speed...

    4. If you are a theme freak, and the visual effect of the IDE is important to you, then Brackets has got you covered. You can choose from a variety of user-made themes, that are installed in just a few seconds. Just go to File -> Extension Manager -> Themes. You can even drag a .zip file and install a theme, or install a theme from an URL.

    When talking about productivity, Brackets has a multi-select option where you can hold CTRL or Command and click on multiple places in the code and the cursor will appear, so you can simultaneously change code, avoiding repetitive actions. I believe other IDEs have this option too, but I had to point it out here.

  4. Lightweight and fast

    5. Since my laptop is not quite powerful and launching, for example, VS Code can take some time, Brackets does the perfect job for me, since it's not as complex as VS Code, but still I manage to do both front-end and back-end with it.


These are just some of the cool stuff Brackets has to offer. If you don't need a lot of plugins and are still just beginning Web Design, I believe Brackets is the perfect fit for you.
