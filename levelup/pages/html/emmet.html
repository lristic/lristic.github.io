Emmet - Code HTML Faster

March 30, 2020 Lazar RistiÄ‡

Today, we will learn about a useful tool that is implemented in all modern IDEs, called Emmet.

It is used to speed up HTML markup coding and shorten the process of creating a base of the HTML. I will go through some basic examples, just so you can get the feel for it.

For this tutorial we will only need one page; index.html.

Emmet is differently used in some IDEs, but the difference usually lies in two buttons; either Enter or Tab. Since I'm using Brackets, I use Tab for Emmet. We will begin by only typing! and pressing Tab. What this will do is generate all the stuff you usually write when starting your HTML file: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

Now, that only took us a second, so we can begin coding the body. Say we need a div with a class called header, a div with an id called main and a div with a class called footer-index. We know that you refer to a HTML class with a dot, e.g. .header and you refer to an id with a hashtag, e.g. #main. So, by typing div.header and pressing Tab (or Enter), div#main + Tab and div.footer-index + Tab generates: 

<div class="header"></div>
<div id="main"></div>
<div class="footer-index"></div>

Fast! Let's go even more in depth. Say we need a div with an id called list and inside that div we have an unordered list with it's list items, particularly 3 of them. In Emmet, that is done like this: div#list>ul>li*3 + Tab (or Enter), which results in: 

<div class="list">
        <ul>
            <li></li>
            <li></li>
            <li></li>
        </ul>
    </div>

You can even put in text using Emmet. Say we need a h1 with a class of main-title which says "Welcome!". With Emmet, we generate this with h1.main-title{Welcome!}:

    <h1 class="main-title">Welcome!</h1>

These are some of the basic ideas of Emmet, meant to speed up your coding. As you get the hang of it, you can play around with it and go into more complex fast coding.
