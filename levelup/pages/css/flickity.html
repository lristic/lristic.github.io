Flickity - building an easy swiping carousel gallery

October 16, 2019 Lazar RistiÄ‡


Today, I will show you how to use Flickity, a library for easy building responsive, swipeable carousels.

In our index.html, we will add the resources we need for Flickity, which you can find on it's official website. You can find the CDN for CSS and JS files on the first page of the website. Also, we will need jQuery and you can find the CDN for it on this website. Just select minified version from jQuery 3.x.

After linking the files, your code should look something like this:

One important thing is, jQuery needs to be linked first, before Flickity.
Now let's get to coding. First, we initialize Flickity with jQuery right after closing body tag:

I added different styles to my carousel cells so I could make them more visible when testing. You can style them in any way. You can put whatever you want inside these divs, depending on your needs.

That is all! You can play around with many options Flickity provides and all of them can be found on their website. I'll show you some of them now.
1. If you want an infinite scroll, meaning that when (in this case) you reach the fifth div, you can continue swiping without going back. This option is called wrapAround and you put it to true in jQuery.
2. Another great option is auto play, called, autoPlay :). You put it's value to number of miliseconds after which you want to switch the carousel-cell.
3. Going hand to hand with the previous one, by default an autoplay stops if you hover over it, so I personally like to turn this off, so it always auto plays. This option is called pauseAutoPlayOnHover and you put it's value to false.

Now, our jQuery code should look something like this:

<script>
    $('.main-carousel').flickity({
        // options
        cellAlign: 'center',
        contain: true,
        autoPlay: 1500,
        pauseAutoPlayOnHover: false,
        wrapAround:true
    });
    
</script>
